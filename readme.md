People - учебный проект адресной книги. Две версии (v1 и v2) должны обладать одинаковым функционалом, но использовать различные инструменты.

Общее описание:
После окна загрузки, появляется список персон, который можно прокручивать. В каждой строке выдается определенный тип данных, например имя+фамилия. Данные отсортированы. При нажатии на элемент списка открывается окно с подробной информацией. Из основного меню можно попасть в меню настроек (?)

список данных:
- имя
- фамилия
- дата рождения
- телефон

возможная дополнительная информация (на перспективу):
- фото
- адрес (текст или место на карте)

Различия по версиям:
| задача | v1 | v2 |
| ----------- | ----------- | ----------- |
| Интерфейс | UIKit | SwiftUI |
| Хранение данных | CoreData | SwiftData |
